HTML Introduction

HTML is the standard markup language for creating Web pages.

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

    • HTML Paragraph

    A paragraph always starts on a new line, and is usually a block of text.

    The <p> Element

    The HTML <p> element defines a paragraph.

    A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

    Lorem, ipsum dolor sit amet consectetur adipisicing elit. Suscipit, excepturi exercitationem incidunt nihil distinctio ad repellat nemo odit fugit, tempore nam aliquam! Iusto, totam ratione. Rerum quia minus error maxime.

    Example for paragraph

    <p> This is my first paragraph.</p>
    <p> This is my another paragraph.</p>

    HTML Div

    The <div> element is used as a container for other HTML elements.

    The <div> Element

    The <div> element in HTML is a generic container used to group elements together. It provides a way to structure and style content on a web page. While it doesn't have a specific meaning by itself, it's often used to create logical divisions within a document.

    Example

    A <div> element takes up all available width:

    <div> This a div element.</div>

    HTML Inline

    Inline elements in HTML are elements that display their content in a single line and do not start a new line. They are typically used for text-based content that should flow within the surrounding text.

    Example for inline

    <h2 style="color: #333;">Student Registration</h2>
